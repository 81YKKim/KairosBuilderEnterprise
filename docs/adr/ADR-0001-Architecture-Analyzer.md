# ADR-0001 Architecture Analyzer Foundation

## Status

Accepted

## Context

Kairos Builder Enterprise is evolving into an Enterprise Software Factory.

To generate software automatically,
Builder must first understand software architecture.

## Decision

Introduce Architecture Analyzer as a core service.

## Responsibilities

- Analyze repository
- Detect architecture
- Produce architecture report
- Supply information to AI generators

## Consequences

- Architecture awareness
- AI planning foundation
- Future dependency analysis
- Automatic architecture validation
